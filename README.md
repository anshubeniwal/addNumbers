# What is this?

Get addition of N Integers.

# Installation

`npm i numbersaddition --save`

Then...

```
import { addNumbers } from 'numbersaddition'

let addition = addNumbers(num1, num2)
```

## Options

addNumbers supports N arguments.

* *num1* - 1,2,3....
* *num2* - 1,2,3....
* *.* - .....
* *.*
* *.*
